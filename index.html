LoL Tracker Community Bot ("the Bot") is an independent Discord bot that tracks ranked League of Legends performance for members of a Discord server. It is not affiliated with, endorsed by, or sponsored by Riot Games.

1. Data We Collect

The Bot collects and stores the following data when a user registers:

  • Riot account information — Summoner name, tag line, and PUUID, obtained via the official Riot Games API.
  • Match history — Ranked match results (champion, KDA, CS, vision score, duration, queue type, timestamp) fetched from the Riot Games API for registered accounts only.
  • Ranked data — Current tier, division, and LP, refreshed periodically via the Riot Games API.
  • Discord user ID — Used solely to mention users in notifications and to link a Discord account to a Riot account.

We do not collect passwords, email addresses, payment information, or any data outside the Riot Games API scope.

2. How We Use the Data

  • Send ranked match result notifications to the server's notification channel.
  • Display leaderboards, statistics, and historical performance on request via bot commands.
  • Generate AI-assisted match analysis using Google Gemini — only anonymized match statistics (champion names, KDA, roles) are sent; no personally identifiable information is included in AI requests.
  • Detect and notify internal matchups between tracked players.
  • Store a per-server language preference (pt_BR or en_US) to localize bot responses. No personal language data is collected.

3. Data Storage

Data is stored in a private PostgreSQL database hosted on Supabase. Access is restricted to the Bot's backend and the server administrator. Data is never sold, rented, or shared with third parties beyond what is necessary for the Bot to function.

4. Third-Party Services

  • Riot Games API — All League of Legends data is sourced exclusively from the official Riot Games API. Use is subject to Riot's Developer Policies.
  • Discord — Notifications are delivered via the Discord API. Use is subject to Discord's Privacy Policy.
  • Google Gemini — Used for optional AI-generated match commentary. Only non-identifiable match statistics are transmitted.

5. Data Retention

Data is retained for as long as the user remains registered in the Bot. Upon removal, the user's record is deactivated. Users may request full deletion of their data at any time.

6. Your Rights

Any registered user may:

  • View their tracked data using bot commands (!elo, !kda, etc.).
  • Remove themselves from tracking by contacting a server administrator.
  • Request deletion of all stored data by contacting the server administrator.

7. Current Usage Model

The Bot is currently provided free of charge to Discord server members. No data is sold or monetized. This project complies with the applicable Riot Games Developer Policies for its current usage tier.

8. Contact

For any privacy-related requests or questions, please contact the server administrator via Discord or open an issue on the project repository.
