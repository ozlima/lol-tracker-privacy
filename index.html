Commands

CommandDescription
👤 Account
!registrar <Name#TAG>Register your LoL account – e.g. !registrar IDV Koala#BR1
!addplayer <Name#TAG>Add a player to the tracker (admin)
!removeplayer <Name#TAG>Remove a player from the tracker (admin)
!players  /  !jogadoresList all registered members
🏆 Ranking
!elos  /  !elosdacomunidadeSolo/Duo ranking of all members
!rank flex / !ranking flexFlex ranking of all members
!rank / !ranking [player]Detailed elo of a player (omit name to see yours)
!top piores / !top melhorCommunity KDA ranking – worst or best players in recent games
📊 Stats
!kda [player]KDA from last 20 ranked games
!mastery / !maestria [player]Top 7 champion masteries
!best / !melhor <role>Top 5 in the community by role (top · jungle · mid · adc · support)
!ingame  /  !partidaShows who is currently in a game
🤖 AI & Analysis
!analyze / !analisar [player]AI profile analysis – strengths, weaknesses and tips
!compare / !comparar <p1> vs <p2>1v1 comparison between two players with AI
!h2h / !confronto <p1> vs <p2>Head-to-head history with AI commentary
!summary / !resumoGenerate the daily community summary
⚙️ Server Admin
!setup #channelSet the notification channel for this server (admin)
!updateSync and verify player nicknames for this server (admin)
!language pt_BR / en_USChange the bot language for this server (admin)
!ajuda  /  !helpShows this command list
